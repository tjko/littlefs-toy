# littlefs-toy
Tool for working with LittleFS images.

## Documentation

A comprehensive manual page is available for the `lfs` command. To view it:

```bash
man ./man/lfs.1
```

Or install it system-wide:

```bash
make install-man
man lfs
```

## Installation

To install the manual page:

```bash
make install-man
```

To uninstall:

```bash
make uninstall-man
```
